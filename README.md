## 💻 Sobre o projeto

Este projeto foi desenvolvido como trabalho escolar, com o objetivo de aplicar na prática conceitos de desenvolvimento web, organização de layout e criação de interfaces responsivas.

Durante o desenvolvimento, foquei em transformar a proposta acadêmica em um projeto real, simulando a criação de um site institucional para uma marca de brigadeiros, fortalecendo minhas habilidades em HTML, CSS e estruturação visual.

---
